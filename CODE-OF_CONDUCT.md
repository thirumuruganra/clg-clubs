# Code of Conduct

## Our Commitment

WAVC is a student-focused open source project. We are committed to a welcoming,
safe, and respectful environment for everyone, regardless of background,
identity, experience level, or institution.

## Expected Behavior

- Be respectful and constructive in discussions.
- Assume good intent and ask clarifying questions.
- Give and receive feedback professionally.
- Focus on improving ideas, not attacking people.
- Support new contributors and help them onboard.

## Unacceptable Behavior

- Harassment, abuse, or discriminatory language.
- Personal attacks, insults, or trolling.
- Sexualized language or unwanted advances.
- Doxxing or sharing private information without consent.
- Repeatedly derailing discussions or acting in bad faith.

## Scope

This Code of Conduct applies to all project spaces, including:

- Issues and pull requests
- Discussions and comments
- Documentation and review feedback
- Any official community channel linked to this project

## Reporting

If you experience or witness unacceptable behavior, report it to project
maintainers through GitHub (issue/discussion) with as much context as possible.

If your report is sensitive, ask for a private response from maintainers rather
than posting details publicly.

## Enforcement

Maintainers may take any action they consider appropriate, including:

- Warning the contributor
- Removing harmful comments/content
- Temporarily restricting participation
- Permanent ban from the project spaces

All reports will be reviewed fairly and handled with care.
